# iteration-1 评测说明

## 目标

验证 `real-search-skill` 是否已经从“说明性草稿”升级为可执行、可复现、可维护的深度研究 skill。

## 方法

本轮评测包含三层：

1. **静态覆盖检查**：检查 `SKILL.md` 是否覆盖 intent capture、workspace setup、execution loop、evidence rules、completion gate。
2. **脚本烟测**：运行初始化、资料索引、笔记生成、质量检查脚本，证明确定性工具可用。
3. **真实案例回放**：把此前 Agent 框架调研压缩成一个 example，用来检查产物结构是否能承载真实长程研究。

## 暂未完成

本轮暂未执行独立子代理 with-skill/baseline 对比。因此 `benchmark.json` 只记录结构化评测结果，不记录 token/time 对比。

## 后续正式评测

下一轮应执行：

- with-skill run：给执行器提供 skill 路径和 eval prompt。
- baseline run：不给 skill，只给原始 prompt。
- 保存 outputs/transcript/timing。

- 聚合 benchmark.json。
- 用 `skill-creator/eval-viewer/generate_review.py` 生成 review 页面。
