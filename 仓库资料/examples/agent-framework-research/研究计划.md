# 研究计划

## 主题

现代 Agent 基座调研：面向算法工程师的 Python 优先、轻量、高自定义、高扩展 Agent runtime。

## 模式

研究落地。

## 目标

- 理解成熟 Agent 框架的核心抽象和运行循环。
- 比较 OpenClaw、Claude Code、OpenAI Agents SDK、LangGraph、OpenHands、SWE-agent、Aider、AutoGen、CrewAI、PydanticAI 等方案。
- 阅读代表性论文：ReAct、Toolformer、Reflexion、Generative Agents、Voyager、MetaGPT、SWE-agent。
- 判断是否已有足够接近的轻量基座。
- 形成一个可进入 MVP 的项目方案。

## 关键问题

1. 成熟框架如何组织 Agent、Runner、Tool、Memory、Trace、Permission？
2. Skills 与 MCP 的关系是什么？
3. 多 Agent 应该如何轻量落地？
4. 如果自研，差异化和最小内核是什么？

## 阶段安排

1. 建立资料索引和候选清单。
2. 对重点框架做统一维度笔记。
3. 对论文做工程映射。
4. 汇总成熟模式和差异化机会。
5. 写项目构思和 MVP 路线。
