# 使用指南

## 触发方式

你可以直接说：

- “帮我深度调研这个领域。”
- “我想学这个方向，但说不清楚，从主流和前沿开始帮我看。”
- “不要只总结，clone GitHub 看源码，论文也看。”
- “帮我从这个模糊想法整理成 MVP 方案。”

## 推荐提示词

```text
开启 real-search-skill。我要研究 [主题]，我现在不太懂，希望你先规划，再搜索主流/前沿/成熟方案，读论文和源码，最后在本地生成一套中文文档。
```

## 默认工作区

```text
深度调研/{日期}-{主题}/
```

## 常见产物

- `研究计划.md`
- `资料索引.md`
- `阶段日志.md`
- `项目调研记录/`
- `论文调研记录/`
- `源码阅读记录/`
- `调研报告.md`
- `落地方案.md`

## 注意

这个 skill 默认只做本地研究和文档，不推送 GitHub、不发布文章、不调用外部写入 API。
