# 轻量高扩展 Agent 基座落地方案

## 项目定位

面向算法工程师的轻量 Agent runtime kernel。它既是学习现代 Agent 方法的源码教材，也是可快速搭建自定义 Agent 的 Python 基座。

## 核心原则

- Python 优先。
- 低封装、高透明。
- Skill 和 MCP 双扩展。
- Trace/eval/permission 从第一版进入。
- 不复刻完整桌面或云平台。

## 核心接口

```python
class AgentSpec:
    name: str
    instructions: str
    tools: list[str]
    skills: list[str]
    max_turns: int

class Runner:
    async def run(self, agent: AgentSpec, task: str): ...
```

## MVP 范围

1. 单 Agent ReAct loop。
2. Python function tools。
3. Markdown Skill 加载。
4. JSONL trace。
5. 本地 MemoryStore。
6. PermissionPolicy。
7. 最小 eval 样例。

## 暂不做

- 完整 GUI。
- 云端协作平台。
- 大而全多 Agent DSL。
- 多模型 provider 全兼容。
